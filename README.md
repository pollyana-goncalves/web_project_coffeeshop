# Triple Espresso

Página de uma cafeteria com vídeos de receitas disponíveis para seus clientes, formulário para reserva de mesa e contatos nas redes sociais.

Desenvolvido como projeto por Pollyana Gonçalves para o Bootcamp de Fullstack da TripleTen.

Como melhoria, acredito que seria interessante ter uma galeria do espaço e uma seção com os diferenciais do local. Outro ponto que poderia ser incluído é a planta baixa ou disposição das mesas para que as pessoas escolhessem os lugares para reserva.
